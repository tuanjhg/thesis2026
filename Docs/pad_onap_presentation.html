04 — gNMI Simulator & NetFlow Collector

Module M1: Streaming Telemetry

gNMI Simulator

  • gRPC/gNMI protocol — port 57400
  • Subscribe mode: STREAM/SAMPLE
  • Interval: 1 second per telemetry push
  • Metrics: bandwidth, PPS, error-rate
  • Attack injection: configurable per scenario

Feature Calibration

f̂ = μ + σ · N(0,1) · attack_mult

Mỗi attack type có calibration riêng để giả lập traffic pattern thực tế từ CICDDoS2019.

NetFlow Collector

  • UDP port 2055 — NetFlow v9
  • Sliding window: 5s window / 1s slide
  • Produce 17 features per window
  • Publish → netflow.features topic

Kafka Producer — Hardened

  • Async send với add_errback
  • Exponential backoff: 2→4→8→…→60s
  • Flush mỗi 5 giây
  • Auto-reconnect on failure

07 — Dataset: Temporal Split & Distribution Shift

Train/Test: Temporal 80/20 — Distribution Shift

Temporal Split Rationale

  • Ngăn data leakage qua thời gian
  • Train: 80% đầu (28,742 windows)
  • Test: 20% cuối (7,186 windows)

09 — Track A: XGBoost Classifier

XGBoost 4-Class — FGSM Adversarial + SHAP

Model Configuration

n_estimators500
max_depth6
learning_rate0.05
tree_methodhist (GPU)
objectivemulti:softprob
num_class4
scale_pos_weightbalanced

5-Fold Stratified CV

  • Oversampling only inside train fold
  • Prevents CV optimism bias
  • Temporal ordering preserved

FGSM Adversarial Augmentation

x' = x + ε · sign(∇ₓ L(f(x), y))
  • ε = 0.01 (perturbation)
  • Gradient from XGBoost leaf weights
  • Augmented samples mixed vào training set
  • Tăng robustness với adversarial attacks

SHAP TreeExplainer

  • Per-prediction explanability
  • Global feature importance ranking
  • Published cùng AI output payload

12 — Track B: Transformer+LSTM Forecaster

Dự báo Tấn công 4-Horizon Proactive

Architecture

  • Input: (batch, 12, 17)
  • Sinusoidal Positional Encoding
  • TransformerEncoder: 2L, 2 heads, d=64
  • LSTM(128) — sequence memory
  • FC → Sigmoid → [P30,P60,P90,P120]

Proactive Trigger Logic

HorizonThresholdAction
t+30s0.70WARN
t+60s0.70PREPARE
t+90s0.70PRE-MITIGATE
t+120s0.70BLOCK
# Forward pass
x = input (B, 12, 17)
x = proj_linear(x) → (B, 12, 64)
x = x + pos_encoding(12, 64)
# Transformer
x = TransformerEncoder(x) → (B, 12, 64)
# LSTM
_, (h_n, _) = LSTM(x) → (B, 128)
# Output heads
out = FC(h_n) → Sigmoid → (B, 4)
# [P_t30, P_t60, P_t90, P_t120]

Training Config

  • Loss: BCE (binary cross-entropy per horizon)
  • Optimizer: AdamW, lr=1e-3
  • Epochs: 50, EarlyStopping(patience=10)
  • Batch: 256, CUDA acceleration
